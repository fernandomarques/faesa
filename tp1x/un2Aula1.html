Introdução

  • A estrutura de dados mais simples de um algorítimo é conhecida como sequencial
  • Os passos são executados um a um na ordem que são especificados
  • Normalmente composta de:
    • Entrada de dados
    • Processamento
    • Saída de dados

Bibliotecas

  • São arquivos contendo várias funções que podem ser incorporadas ao programa
  • A diretiva #include adiciona a biblioteca no programa
  • A biblioteca stdio.h permite a utilização de comandos de entrada e saída
  • É importante salientar que a linguagem C é sensível (case-sensitive) a letras maíusculas e minúsculas
  • 
#include <nome_da_biblioteca>
int main(){
	bloco_de_comandos;
	return 0;
}

Comando de Saída

  • São usados para mostrar dados na tela
  • 
printf("Ola!);
printf(" Esta funcionando!!!");

Declaração de Variáveis em C

  • As variáveis são declaradas após a especificação de seus tipos
  • Os tipos mais usados são int, float e char
  • Também existem os tipos double e bool!
  • Não existe um tipo para cadeias de caracteres
  • int x;
float x,h;
char sexo;

Declaração de Variáveis em C

  • Também podemos inicializar as variáveis
  • 
int x =1;
float y= 1.5, z= 100.25;
char sexo= 'M';
  • Importante antes de usar uma variável devemos declarar-la!

Tipos de Dados

  • char são números
  • long para números inteiros bem grandes
  • short para números pequenos e inteiros
  • int para a maioria dos números inteiros
  • int tem tamanhos diferentes em máquinas diferentes
  • float para a maioria dos números de ponto flutuante
  • double para números de ponto flutuante bem precisos

Declaração de Constantes em C

  • São declaradas após os #include
  • Seus valores não podem ser alterados durante a execução
  • 
#define x 7
#define y 4.5
#define nome "Fernando"
  • Também podemos usar o modificador const
  • const int x = 7;

Comando de Atribuição em C

  • O comando de atribuição é o sinal =(sinal de igualdade)
  • x = 4;
x = x+ 2;
y = 2.5;
sexo = 'F';
  • 'Caracter' e "String"
  • Cada comando é finalizado com ;
  • A parte inteira e fracionária são separadas por ponto

Comando de Saída

  • São usados para mostrar dados na tela
  • 
printf("%i", valor);
printf("%f", x);
printf("Conteúdo de y = %i", y);
printf("%5.2f", x);

Comando de Saída

  • É necessário informar o tipo da variável
  • %f float, %d ou %i inteiro, %c caracter
  • Para adicionar uma quebra de linha, usamos \n
  • Para uma tabulação \t
  • 
printf("Aula\n");
printf("\tFácil");
