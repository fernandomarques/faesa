## Objetivos da aula Nessa aula espera-se que os alunos desenvolvam: - Utilização de IDE para desenvolvimento de Python - Conceitos básicos de programação em python com chamadas de função; - Conhecer a sintaxe básica da linguagem Python - Compreensão dos comandos básicos da biblioteca turtle, como foward(), backward(), left() e right(); - Capacidade de interpretar e seguir instruções de desenho; - Habilidade para realizar ajustes e correções em desenhos;
## Descrição Detalhada da Atividade - Introdução (10 minutos) - Criação do Primeiro desenho (15 minutos) - Criação de desenhos mais complexos (50 minutos) - Compartilhamento dos desenhos (20 minutos) - Conclusão
## Expectativa de Desempenho do aluno - Ser capaz de criar desenhos simples utilizando a biblioteca Turtle - Ser capaz de entender os conceitos básicos de biblioteca e chamada de função - Utilizar comandos simples como forward, backward, right, left, penup and pendown. - Aplicar os conhecimentos aprendidos para criar desenhos criativos e esteticamente agradáveis - Compartilhar os desenhos e explicar como foram feitos
