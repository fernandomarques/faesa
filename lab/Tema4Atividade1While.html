Acumuladores e Contadores

  • Um acumulador é uma variável que ocorre em ambos os lados de uma atribuição
  • Antes de ser usado deve ser iniciado.
  • Como acumuladores são muito comum existem operadores aritméticos de atribuição 
  • 
	a = 2
	a +=2 # a = a + 2
	b=0
	b++  # b = b + 1 # NAO FUNCIONA EM PYTHON
## Contadores - Sempre são do tipo __inteiro__! - Devem ser __inicializados__ antes de usar `cont = 0` - A operação mais comum é `cont++` ou `cont+=1` - Normalmente são colocados dentro do loop
## Acumuladores - Sempre são do __tipo da variável__ que está sendo acumulada! - Devem ser __inicializados__ antes de usar, normalmente com o valor 0 - A operação mais comum é `cont+=var` - Normalmente são colocados dentro do loop - Em alguns casos devemos fazer operação após o loop, exemplo: média

Repetição com terminação forçada

  • Quando usamos while true: ... criamos uma repetição infinita
  • Para forçar o término dessa execução podemos usar o comando break
  • Não é convencional utilizar e deve-se ter cuidado ao usar
  • Dada uma sequencia de números positivos informe o total a ser pago. Quando for digitado o valor 0, pare de ler!

Estrutura de repetição com pós condição

  • Com pós-condição podemos executar um ou vários comandos.
  • É usada quando devemos executar ao menos uma vez
  • // Java
do{
	comando1;
	comando2;
} while(condicao);
  • # Python
while True:
	comando1
	comando2
	if condicao:
		break
