Utilizando for

  • O comando for possui 3 campos
    1. Inicialização da variável de contagem
    2. Condição de parada
    3. Atualização da variável

Exemplo For


{
	int i;
	for(i=0; i < 10 ; i ++){
		System.out.println(i);
	}
}
  1. Façam um código que conte de 1 até 6 imprimindo na tela
  2. Façam um código que conte de 6 até 1 imprimindo na tela

Laços for aninhados


{
	int i,j;
	for(i=0; i < 10 ; i ++){
		for(j = 9; j >= 0 ; j--)
		System.out.println(i + " " +j);
	}
}
