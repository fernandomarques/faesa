Uma curiosidade fascinante sobre Python é sua filosofia de design, conhecida como "The Zen of Python" (O Zen do Python). Esse conjunto de princípios foi escrito por Tim Peters e encapsula os valores e ideais subjacentes à linguagem. Alguns dos princípios incluem "Clareza é melhor que sutileza", "Simples é melhor que complexo" e "Legibilidade conta". Essa filosofia ajuda a orientar os desenvolvedores na escrita de código Python de maneira eficaz e elegante. Para visualizar o "The Zen of Python", basta digitar import this no interpretador Python.

Exemplo Entrada de dados - OOOOPS

import java.util.*;
public class Exemplo{
	public static void main(String[] args){
		Scanner scan = new Scanner(System.in);
		System.out.print("Informe seu nome");
		String nome = scan.next()
		System.out.print(nome);
	}
}

Exemplo Entrada de dados

nome = input("Informe seu nome")
print("Olá ", nome)

Lendo Valores Numéricos

idade = input("Qual a sua idade?")
print(idade)
print(type(idade)) # input sempre acha que é str

idade = int(idade) # converte para inteiro
print(idade)
print(type(idade))

ano = int(input("Que ano você nasceu?"))
idade = 2023 - ano
print(idade)
