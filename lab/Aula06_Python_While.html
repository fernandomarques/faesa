Escreva um programa em Python que solicita ao usuário um número inteiro positivo N e, em seguida, imprime todos os números pares menores ou iguais a N.

Pergunta: Qual é a expressão condicional utilizada no programa para verificar se um número é par?

  1. n % 2 == 0
  2. n % 2 != 0
  3. n % 2 > 0
  4. n % 2 < 0
  5. n % 2 == 1

## Erros comuns em loop 1. Esquecer de inicializar a variável de controle. 1. Esquecer de incrementar/decrementar a variável de controle. 1. Esquecer de definir a condição de término do loop. 1. Modificar a variável de controle dentro do loop. 1. Não lidar com entradas inválidas. 1. Não definir a sequência correta para o loop. 1. Não usar a estrutura correta de loop para o problema em questão.
