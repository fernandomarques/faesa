Arrays Numpy

NumPy é uma biblioteca de Python para manipulação eficiente de arrays multidimensionais e funções matemáticas.

import numpy as np

vetor1 = np.array([1, 2, 3, 4, 5])
vetor2 = np.arange(0, 10, 2)
vetor3 = np.zeros(5)  # Vetor de 5 elementos preenchidos com zeros
vetor4 = np.ones(3)  # Vetor de 3 elementos preenchidos com uns
vetor5 = np.random.rand(4)  # Vetor de 4 elementos com valores aleatórios entre 0 e 1
vetor6 = np.random.randint(0,20, size=10)  # Vetor de 4 elementos com valores aleatórios entre 0 e 1

Arrays Numpy

  1. numpy.array: Cria um array multidimensional com elementos homogêneos.
  2. numpy.shape: Retorna as dimensões do array.
  3. numpy.reshape: Altera a forma do array sem alterar seus dados, permitindo reorganizar os elementos.
  4. numpy.mean: Calcula a média dos elementos do array ao longo do eixo especificado.
  5. numpy.max: Encontra o valor máximo no array ao longo do eixo especificado.

Além disso também é possível fazer operação de multiplicação, soma e etc. de forma nativa
