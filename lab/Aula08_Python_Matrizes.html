## Dicionários em Python - Um dicionário é uma estrutura de dados poderosa em Python. - Permite armazenar pares chave-valor. - As chaves são únicas e imutáveis. - Os valores podem ser de qualquer tipo.

Exemplo de Dicionário

### Criando um dicionário de filmes
filmes = {
	"Ação": ["Vingadores", "Missão Impossível"],
	"Comédia": ["Se Beber, Não Case!", "Ace Ventura"],
	"Drama": ["O Poderoso Chefão", "Cidade de Deus"]
}

# Acessando valores do dicionário
print(filmes["Ação"])  # Saída: ["Vingadores", "Missão Impossível"]
print(filmes.get("Comédia"))  # Saída: ["Se Beber, Não Case!", "Ace Ventura"]
## Tuplas em Python As tuplas são uma estrutura de dados em Python semelhantes às listas, mas com uma diferença fundamental: elas são imutáveis. Isso significa que uma vez criada, uma tupla não pode ser alterada. - As tuplas são definidas utilizando parênteses (). - Podem conter elementos de diferentes tipos de dados. - Os elementos são acessados pelo índice, assim como em listas. - Tuplas podem ser concatenadas e fatiadas.
## Diferença entre Tuplas e Listas Tuplas: - Imutáveis - Utilizam parênteses () - Mais eficientes em espaço e tempo - Podem ser usadas como chaves de dicionários Listas: - Mutáveis - Utilizam colchetes [] - Permitem operações de alteração - Mais flexíveis para manipulação de dados A escolha depende do contexto e das necessidades do programa.
## Vantagens das Tuplas 1. Imutabilidade: A imutabilidade das tuplas garante que os dados não sejam alterados acidentalmente. 2. Desempenho: As tuplas são mais eficientes em termos de espaço e tempo de execução do que as listas. 3. Integridade dos Dados: Como as tuplas são imutáveis, elas garantem a integridade dos dados, evitando alterações indesejadas. 4. Utilização como Chaves de Dicionários: Tuplas podem ser usadas como chaves de dicionários, pois são imutáveis e garantem a unicidade dos dados. 5. Iteração Eficiente: É mais eficiente percorrer elementos em tuplas do que em listas, especialmente para grandes conjuntos de dados. 
# Definindo uma tupla
frutas = ('maçã', 'banana', 'laranja', 'uva')

# Acessando elementos da tupla
print(frutas[0])  # Saída: maçã
print(frutas[2])  # Saída: laranja

# Tentativa de modificação (gera erro)
frutas[1] = 'morango'

# Iterando sobre uma tupla
for fruta in frutas:
	print(fruta)

# Verificando se um elemento está na tupla
if 'uva' in frutas:
	print("A uva está na lista de frutas.")

# Obtendo o tamanho da tupla
print(len(frutas))  # Saída: 4
## Matriz com Numpy - A biblioteca NumPy oferece suporte para trabalhar com matrizes multidimensionais. - Uma matriz em NumPy é uma tabela de elementos, todos do mesmo tipo, indexados por um tuplo de inteiros não negativos. 

# Exemplo de criação de uma matriz:

import numpy as np

# Criando uma matriz 2x3
matriz = np.array([[1, 2, 3], [4, 5, 6]])

# Imprimindo a matriz
print(matriz)
## Operações com Matrizes - NumPy permite realizar diversas operações com matrizes. - É possível realizar operações aritméticas, transposição, multiplicação de matrizes, entre outras. 
import numpy as np

# Criando matrizes
matriz1 = np.array([[1, 2], [3, 4]])
matriz2 = np.array([[5, 6], [7, 8]])

# Operações aritméticas
soma = matriz1 + matriz2
subtracao = matriz1 - matriz2
multiplicacao = matriz1 * matriz2
divisao = matriz1 / matriz2

# Imprimindo resultados
print("Soma:")
print(soma)
print("Subtração:")
print(subtracao)
print("Multiplicação:")
print(multiplicacao)
print("Divisão:")
print(divisao)
