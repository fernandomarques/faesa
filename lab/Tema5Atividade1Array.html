Array ou Vetores

Created by Fernando Marques

O que são vetores

  • Vimos exercícios de calcular média de notas, mas e se fossem 100 notas? Como declarar as variáveis?
  • Podemos salvar um número pré-determinado de valores

Definição

  • Vetor é uma variável composta homogênea unidimensional
  • É um conjunto de variáveis do mesmo tipo
  • Possuem o mesmo identificador e são alocados sequencialmente
  • O índice indica a posição dentro do vetor

Declaração

  • É capaz de armazenar uma coleção de dados do mesmo tipo
  • Cada dado armazenado em um vetor é chamado item ou elemento
  • O índice começa do valor 0
  • Para dizer que uma variável é um vetor devemos colocar o sufixo [n], onde n é seu tamanho
  • 
int[] valores = new int[3];
float[] precos = new float[3];

Acessando elementos e índices

  • Depois de declarar o vetor, podemos acessar seus elementos
  • Para atribuir o valor 10 ao 21º elemento: notas[20]= 10;
  • O índice do primeiro elemento é o 0
  • Se tentar acessar um índice maior que o tamanho do vetor teremos um erro!

Inicializar vetor


{
int notas[] = { 1,2,3};
// ou
int notas[] = new int[] { 1,2,3};
}

Acessando elementos do vetor com for


{
	int i;
	int v[] = new int[10];
	for(i=0; i < v.length; i++){
		v[i];
		System.out.println(i);
	}
}
