## Objetivos - Comportamento assintótico de funções; A notação O, ômega e theta;
- Vimos que seria possível calcular exatamente quanto tempo um algoritmo levaria para ser executado - Porém, quando vamos comparar, é suficiente saber que o seu \Theta(n), já que para números grandes os "por menores" são irrelevantes - A ordem de crescimento do tempo de execução é conhecida como eficiência assintótica - Veremos algumas notações e faremos uma revisão de algumas funções comuns
## Notação Assintótica - Usaremos a notação assintótica primariamente para definir o tempo de execução do algoritmo - Vimos que no pior caso o insertionSort é $ \Theta(n^2) $ - Na realidade a notação assintótica é aplicada em funções - Caracterizamos o tempo no pior caso como $an^2 + bn + c$ para constantes a,b e c - Ao afirmar que o tempo de execução é de $ \Theta(n^2)$ estamos abstraindo alguns detalhes da função - Como a notação assintótica se aplica a funções $ \Theta(n^2)$ era a função $an^2 + bn + c$ que, por acaso, caracteriza o tempo de execução no pior caso
## Notação Θ - Vimos que o tempo de execução no pior caso do InsertionSort foi de $T(n) = \Theta(n^2)$ - Para uma dada função $g(n)$, denotamos por $\Theta(g(n))$ o conjunto de funções: $$ \Theta(g(n)) = [ f(n) \exists c_1,c_2,n_0 $$ $$ 0 \leq c_1g(n) \le f(n) \le c_2g(n), \forall n \ge n_0 ] $$ - _a notação correta é {} mas a fórmula não aceitava..._
## Notação Θ - Uma função f(n) pertence ao conjunto $\Theta(g(n))$ se existirem constantes positivas $c_1,c_2$ tais que ela possa ser encaixada entre $c_1g(n)$ e $c_2g(n)$ para um n suficientemente grade - Como $\Theta(g(n))$ é um conjunto poderíamos escrever $ f(n) \in \Theta(g(n))$ para indicar que f(n) é um membro de $\Theta(g(n))$ - Dizemos então que g(n) é um __limite assintoticamente restrito__ para f(n) - A definição de $\Theta(g(n))$ exige que todo membro seja assintoticamente não negativo, ou seja, que seja positiva para todo _n_ suficientemente grande
## Notação Θ - Vimos a notação Θ de forma informal dizendo que bastava descartar os termos de ordem mais baixa e ignorar o coeficiente do termo de ordem maior - Vamos mostrar que $\frac{1}{2} n^2 - 3n = \Theta(g(n^2))$
$$ c_1, c_2, n_0 $$ tal que $$c_1n^2 \le \frac{1}{2}n^2 -3n \le c_2n^2 $$ Dividindo por $n^2$ $$ c_1 \le \frac{1}{2} - \frac{3}{n} \le c_2 $$

Se $c_1 = 1/14$ e $c_2 = 1/2$ para n=7 podemos verificar que $\frac{1}{2} n^2 - 3n = \Theta(g(n^2))$

Podemos também mostrar que $6n^3 \neq \Theta(g(n^2))$

Suponha $c_2$ e $n_0$ tais que $6n^3 \le c_2n^2$ se dividirmos por n²

$$ n \le \frac{c_2}{6}$$ o que não é válido para um valor arbitráriamente grande de n já que $c_2$ é constante

Também conseguimos demostrar que os termos de ordem mais baixa podem ser ignorados

Considere qualquer função quadrática $ f(n) = an^2 + bn + c$ onde $a,b,c$ são constantes e a > 0

Teremos $ f(n) = \Theta(n^2) $ e podemos verificar que

$ 0 \le c_1n^2 \le an^2 + bn + c \le c_2n^2 $

Podemos afirmar então que para todo $p(n) = \sum_{i=0}^d a_in^i $ temos $$p(n) = \Theta(n^d) $$

Tendo em vista, ainda, que qualquer constante é um polinômio de grau 0 podemos expressar qualquer função constante como $\Theta(n^0)$ ou $\Theta(1)$

Essa notação é um pequeno abuso pois não indica qual variável está tendendo a infinito, mas a usaremos com frequência para indicar uma constante ou função constante em relação a uma variável
