Introdução

  • Uma expressão condicional é uma expressão cujo valor pode ser falso ou verdadeiro
  • Em C, falso é representado por 0 e verdadeiro é representado por diferente de 0
  • Existe também a biblioteca stdbool com os valores true e false
  • Uma condição simples é uma expressão condicional cujo valor é dado por um operador relacional
  • Uma condição composta é uma expressão condicional formata por duas ou mais condições simples ligadas por operadores lógicos
  • (a + 1 > 2*b;)
(2*c == 6)

Lógica em C - Operador Relacional

Operador Operação Exemplo
> Maior 1 > 0
>= Maior ou igual 1 >= 1
< Menor 1 < 2
<= Menor ou igual1 <= 1
== Igual 1 == 1
!= Diferente 1 != 0

Estrutura Condicional em C

  • Estrutura condicional simples
  • O comando só será executado se a condição for verdadeira
  • 
if(condicao)
	comando;
if(condicao){
	comando1;
	comando2;
	comando2;
 }

Estrutura condicional Composta

  • Se a condição for verdadeira será executado o comando 1, se for falso, será executado o comando 2
  • 
if(condicao)
	comando1;
else
	comando2;
if(condicao){
	comando1;
	comando2;
}else{
	comando3;
	comando4;
}

Problema 1

Em uma empresa, paga-se R$ 19,50 por hora trabalhada e desconta-se 25% de imposto de renda dos salários superiores a R$ 2.500,00. Dado o número de horas trabalhadas por um funcionário dessa empresa, informe o valor do desconto de IR (se houver) e o valor do salário a ser pago ao funcionário.

Exemplo: um funcionário trabalhando 40 horas por semana totaliza no fim de um mês 160 horas trabalhadas. Então seria 160 horas * 19,50 valor da hora = R$ 3.120,00. Logo o desconto dele será de R$ 780,00. O salário a ser pago para este funcionário será de R$ 2.340,00.
