Comando de saída

  • Printf
  • Primeira parte entre ""
  • Colocar mensagem
  • Depois de "" e virgula, podemos colocar as variáveis a serem impressas
  • #include <stdio.h>
int main(){
	printf("Ola\n");
	printf("Ola %d\n",3);
	printf("Ola %f\n", 3.3);
	printf("Ola %c \n",'s');
	printf("Ola %d %f %c \n",3,3.3,'s');
}

Comando de entrada

  • scanf
  • Primeira parte entre ""
  • Colocar o tipo da variavel
  • Depois de "" e virgula, podemos colocar as variáveis a serem impressas
  • No scanf precisamos do &
  • #include <stdio.h>
int main(){
	int a;
	float x;
	char chr;
	scanf("%d",&a);
	scanf("%f", &x);
	scanf("%c",&chr);
	scanf("%d %f %c", &a, &x, &chr);
}
