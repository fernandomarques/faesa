Introdução

  • Trechos de código que precisam ser repetidos são chamados de laços de repetição
  • A estrutura de repetição é importante para resolver problemas computacionais como leitura de muitos dados e manipular elementos de matriz
  • Os laços de repetição ou loop fazem uma execução finita em círculos que depois segue o curso normal

Acumuladores e Contadores

  • Um acumulador é uma variável que ocorre em ambos os lados de uma atribuição
  • Antes de ser usado deve ser iniciado.
  • Como acumuladores são muito comum existem operadores aritméticos de atribuição
  • main(){
   int a,b;
	 a = 2;
	 a +=2; // a = a + 2
	 b=0;
	 b++;  // b = b + 1
					}

  • Dado um número natural n, desenha um quadrado n x n. Dica: printf("%d ", 223);

Estrutura de repetição com condição

  • Serve para executar um ou vários comandos enquanto uma determinada condição for verdadeira
  • A condição é avaliada antes de o comando ser executado
  • while(condicao){
   comando1;
	 comando2;
 }

Estrutura de repetição com condição

  1. Dado um número inteiro qualquer, exiba seus dígitos
  2. Em um banco, as contas são identificadas por um número de conta com dígito verificados. Esse dígito é calculado pelo seguinte modo: Somar todos os dígitos do número da conta, depois dividis a soma por 10 e tomar o resto. Exemplo 5713 temos: 5+7+1+3 = 16 dividido por 10 o resto é 6

Repetição com terminação forçada

  • Quando usamos while(1){...} criamos uma repetição infinita
  • Para forçar o término dessa execução podemos usar o comando break
  • Não é convencional utilizar e deve-se ter cuidado ao usar
  • Dada uma sequencia de números positivos informe o total a ser pago. Quando for digitado o valor 0, pare de ler!

Estrutura de repetição com pós condição

  • Com pós-condição podemos executar um ou vários comandos.
  • É usada quando devemos executar ao menos uma vez
  • do{
   comando1;
	 comando2;
 } while(condicao);

Resumo

contar++ significa aumentar contar em 1.
contar-- significa diminuir contar em 1.
Loops FOR são uma maneira mais compacta de escrever loops.
Loops WHILE repetem o código, contanto que a condição seja verdadeira.
Loops DO-WHILE executam o código pelo menos uma vez.
Os laços de repetição são conhecidos também por sua tradução em inglês: loops
