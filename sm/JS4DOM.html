DOM - Document Object Model

  • Dividido em três partes
    • Core DOM – modelo padrão para qualquer documento estruturado
    • XML DOM – modelo padrão para documentos XML
    • XML DOM – modelo padrão para documentos HTML
  • Dom define os objetos e as propriedades de todos os elementos de um documento, assim como os métodos (interface) para acessá-los

DOM

  • Pode ser utilizado para HTML ou para XML
  • É um padrão W3C
  • HTML DOM é um modelo para HTML
  • Independente de linguagem e plataforma
  • Define um padrão para a manipulação de documentos HTML
  • Define um padrão para acessar objetos de um documento XML
  • “Parser de arquivos Estruturados”

DOM

  • Vantagem
    • Possui API robusta para a manipulação dos dados
  • Desvantagem
    • Consumo de memória, pois armazena todo o documento em memória em forma de árvore
    • Não recomendado para arquivos grandes

Árvore DOM

  • Dom é utilizado para a manipulação de arquivos
  • Carrega os arquivos em forma de árvore na memória
  • Todos os elementos são manipulados como nós da árvore
  • Relação Hierárquica entre os nós
    • Pais, filhos, vizinhos, raiz e nós folhas

Árvore DOM HTML

Propriedades DOM

  • nodeName – contém o nome de um nó
  • value – contém o valor (conteúdo) de um nó
  • parentNode – o pai de um nó
  • childNode – os nós filhos de um nó
  • attributes – os atributos de um nó
  • innerHTML – contém o texto

HTML DOM

  • Elementos HTML como objetos
  • Propriedades de todos elementos HTML
  • Métodos de acesso a todos elementos HTML
  • Eventos de todos elementos HTML

Javascript e DOM

  • Javascript consegue criar HTML dinâmico:
    • Modificar todos elementos HTML
    • Modificar todos atributos HTML
    • Modificar todos styles CSS
    • Remover elementos e atributos HTML
    • Adicionar novos elementos e atributos HTML
    • Reagir a eventos HTML
    • Criar novos eventos HTML

Exemplo


<html>
<body>
	<p id="demo"></p>

Criar novos elementos HTML


<div id="div1">
	<p id="p1"> Isso é um paragrafo. </p>
	<p id="p2"> Isso é outro paragrafo. </p>
</div>
<script>
	var para = document.createElement("p");
	var node = document.createTextNode("Isso é novo.");
	para.appendChild(node);

	var element = document.getElementById("div1");
	element.appendChild(para);
</script>

Remover um elemento HTML


<div id="div1">
	<p id="p1"> Isso é um paragrafo. </p>
	<p id="p2"> Isso é outro paragrafo. </p>
</div>
<script>
	var element = document.getElementById("div1");
	var child = document.getElementById("p1");
	parent.removeChild(child);
</script>
