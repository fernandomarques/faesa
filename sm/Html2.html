Tags e Atributos

  • Nós vimos como são as tags em html
  • Atributos são valores da tag
  • Exemplo: 
    <font color='red'></font>

Comentários

Tag font

  • Não é suportado em html 5! O padrão é usar css para mudar cor
  • Atributos
    • size= "3"
    • color= "red"
    • face = "verdana,arial,sans-serif"
  • Façam testes com esse três atributos

Hiperlinks

  • Tag= 
    <a></a>
  • Atributo= 
    <a href="meulink.com"></a>
  • Por padrão:
    • Links não visitados são azuis e sublinhado
    • Links visitados são roxos e sublinhados
    • Links ativo é sublinhado e vermelho

Links - Atributos

  • download (html5)- indica que o link vai ser baixado
  • href - url do link
  • hreflang - linguagem do documento
  • media (html5) - indica para qual midia
  • rel - relação com o documento atual e o linkado
  • target - onde abrir o documento (parent, self, top)
  • type - tipo de mídia
  • Mais detalhes e tags
  • Façam experimentos com os links! Crie link para outras páginas.

Âncoras

  • São utilizadas para acessar partes de uma página
  • Novamente vamos utilizar a tag a
  • Com o mesmo atributo href!
  • no href vamos colocar o id de uma âncora
  • <h1 id="meuTitulo">Titulo</h1>
  • <a href="#meuTitulo">Link para titulo</a><
  • <a href="minhapagina.html#meuTitulo">Link para titulo</a><
  • Mais detalhes
  • Tentem colocar ids e usar ancoras pela página

Barras Horizontais

  • Tag hr
  • Em HTML5: Representa uma quebra temática
  • Em HTML4.01 Representa uma régua horizontal
  • Todos atributos foram removidos em html5!

Imagens

  • Permite adicionar uma imagem à página web
  • Caminho relativo vs. caminho real
  • Tag e atributos:
    <img src="smiley.gif" alt="Smiley face" height="42" width="42">
  • Align, border, hspace, vspace não são mais suportados em html5
  • Link para mais infomações

Imagens Atributos

  • alt - texto para a imagem (caso a imagem não apareça)
  • height - altura em pixels
  • longdes - descrição detalhada da imagem
  • src - caminho da imagem
  • width - largura em pixels
