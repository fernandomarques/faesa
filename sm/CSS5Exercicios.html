Sistemas Multimidia

  • Boas Práticas e Exercício

Created by Fernando Marques

Simplicidade

  • Apesar de beleza ser importante, o usuário está no seu site pela funcionalidade
  • Adicionar elementos que não possuem funcionalidade deixa a navegação no site mais complexa
  • Cores:
    Manda utilizar no máximo 5 cores diferentes (mais ou menos 2)
  • TypeFace: Fontes devem ser legíveis. Utilizar no máximo 03 tipos com 03 tamanhos diferentes
  • Exemplo

Hierarquia Visual

  • Fazer com que a parte importante chame mais atenção
  • O objetivo é induzir o usuário a fazer uma ação
  • Podemos usar uma coloração diferente para chamar a atenção a um elemento
  • Exemplo Spotify

Navegabilidade

  • Deve ser trivial navegar no seu site
  • Colocar a estrutura primária de navegação no topo
  • Adicionar navegação ao rodapé
  • Utilize breadcrumbs
  • Não fornecer muitas opções de navegação
  • Não utilizar mais do que 3 níveis de navegação
  • Reutilizar o mesmo menu
  • Exemplo

Consistência

  • Além da navegação consistente. A cor de fundo, cores, fontes e etc. devem ser consistêntes
  • O layout não precisa se exatamente igual
  • Exemplo: Página de ajuda airbnb

Acessibilidade

  • O site deve ser compatível com diversos dispositivos
  • Exemplo responsividade
  • Coisas simples como legenda em imagem podem ajudar
  • Sites de governo também possuem opções de acessibilidade para deficiêntes visuais

Convenção

  • Algumas convenções se tornam familiares aos usuários
  • Barra de navegação principal superior or esquerda
  • Logo superior esquerdo ou centralizado
  • Logo clicável que redireciona para a página principal
  • Links que mudam de cor ao passar o mouse sobre
  • Exemplo de convenção para site de compra

Credibilidade

  • Convenções de site podem dar mais credibilidade ao site
  • Ser direto e honesto com o produto do site
  • Ter uma página com preços
  • Exemplo de pricing

Dos and Don'ts

  • Utilize convenções comuns
  • Não faça o usuário esperar por conteúdo
  • Utilize uma ordem lógica para seu layout
  • Não encha seu layout com conteúdo em excesso
  • Utilize whitespace e design minimalista
  • Não utilize muitas cores
  • Desenvolva o site com mobile em mente
  • Não utilize tamanhos e fontes de forma errada. Caixa-Alta, Serif, itálico/negrito
  • Não utilize muitas fontes
