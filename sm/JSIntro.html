Onde Colocar o JavaScript

  • Dentro do HTML o js pode estar dentro de head ou body. Sempre dentro da tag script
  • JavaScript pode funcionar como função (sendo chamado diretamente) ou como evento, sendo chamado quando algo acontece
  • É indicado colocar script no final do body, pois a compilação de js pode deixar o carregamento da página lento
  • Também podemos criar um arquivo .js e colocar o atributo src na tag script para adicionar o código

Escrevendo na Tela

  • Usar um alertbox com window.alert()
  • Escrever no documento com document.write() (Apaga o conteúdo que já existe)
  • Escrever dentro de um elemento html (document.getElementById) atualizando seu innerHTML
  • Escrever no console do navegador com console.log (F12)

Sintaxe

  • Comandos são separados por ;
  • Strings podem ser usadas com aspas dupla ou simples
  • Para declarar variáveis utilizamos var
  • Comentários podem ser feitos com // ou /**/
  • JavaScript é case-sensitive!

Expressões e Variáveis

  • Podemos somar strings, tentem!
  • var helloWorld = "Hello" + " " + "World";
  • Mas o que acontece nos exemplos a baixo
  • var x = 2 + 3 + "5"
  • var x = "5" + 2 + 3;
  • Além dos operadores de comparação comuns também temos === e !==. Igual em valor e tipo!
  • Para saber um tipo podemos chamar typeof()

Tipos de dados
