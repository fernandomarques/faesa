O que é flex?

  • Teve seu primeiro working draft em 2009
  • Virou W3C Candidate Recomendation em 2017
  • É suportado pelos principais navegadores
  • Pode expandir itens para ocupar espaços vazios
  • Ou encolher para caber

Container vs Itens

Container


.container {
	display: flex;
	flex-direction: row | row-reverse | column | column-reverse;
	flex-wrap: nowrap | wrap | wrap-reverse;
	flex-flow: <‘flex-direction’> || <‘flex-wrap’>
}

Container 


.container {
	display: flex;
	justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
	align-items: stretch | flex-start | flex-end | center | baseline;
	align-content: flex-start | flex-end | center | space-between | space-around | stretch;	
	}

Items


.item {
	order: <integer>; /* default is 0 */
	flex-grow: <number>; /* default 0 */
	flex-shrink: <number>; /* default 1 */
	// Tamanho padrao do elemento, auto = width, height
	flex-basis: <length> | auto; /* default auto */
	flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]
	align-self: auto | flex-start | flex-end | center | baseline | stretch;
}

Exemplos

  1. Magica em Centralizar (margin auto)
  2. Itens de tamanho fixo, espalhados horizontalmente (row wrap)
  3. Navegacao

  4. Site com cabeçalho, main, rodape e laterais
    • Celular - Em linhas
    • Tablet - Laterais dividem espaço 600px
