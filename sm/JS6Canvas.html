Canvas - O que é?

  • É um elemento HTML5 usado para desenhar em tempo real usando javascript
  • O canvas é apenas um container, precisamos de script para desenhar nele
  • Existem diversos métodos para desenhar linhas, caixas, círculos, textos e etc.

Canvas - Criando

  • 
<canvas width="40%" height="40%" > </canvas>
  • Mas não apareceu nada! Podemos adicionar uma borda só para ter certeza que o canvas está lá...
  • 
<style>
	canvas {
		border: 1px solid black;
	}
</style>
  • Agora sim!

Canvas - Desenhando

  • Para desenhar precisamos de usar javascript, é aconselhado desenhar no evento de carregar o documento
  • 
<tscript&g
	$(window).load(function(){ // Evento de carregar
		var canvas = $("#myCanvas").get(0); // Recuperar o canvas
		var ctx = canvas.getContext("2d"); // Temos que utilizar um context
		ctx.fillRect(0,0,50,50);
	}
</script>

Canvas - Desenhando Linhas

  • 
<tscript>
	$(window).load(function(){ // Evento de carregar
		var canvas = $("#myCanvas").get(0); // Recuperar o canvas
		var ctx = canvas.getContext("2d"); // Temos que utilizar um context
		ctx.strokeStyle = "blue";
		ctx.moveTo(15,15);
	  	ctx.lineTo(200-15,100-15);
	  	ctx.stroke();
	}
</script>

Canvas - Desenhando Círculo

  • 
<tscript>
	$(window).load(function(){ // Evento de carregar
		var canvas = $("#myCanvas").get(0); // Recuperar o canvas
		var ctx = canvas.getContext("2d"); // Temos que utilizar um context
		ctx.strokeStyle = "blue";
		ctx.beginPath();
		ctx.arc(95,50,40,0,2*Math.PI);
		ctx.stroke();

	}
</script>

Canvas - Desenhando Texto

  • 
<tscript>
	$(window).load(function(){ // Evento de carregar
		var canvas = $("#myCanvas").get(0); // Recuperar o canvas
		var ctx = canvas.getContext("2d"); // Temos que utilizar um context
		ctx.strokeStyle = "blue";
		ctx.font = "30px Arial";
		ctx.textAlign = "center";
		ctx.strokeText("Hello World",canvas.width/2,canvas.height/2);

	}
</script>

Canvas - Pintando Gradiente

  • 
<tscript>
	$(window).load(function(){ // Evento de carregar
		var canvas = $("#myCanvas").get(0); // Recuperar o canvas
		var ctx = canvas.getContext("2d"); // Temos que utilizar um context
		var grd=ctx.createLinearGradient(0,0,25,25);
	        grd.addColorStop(0,"red");
	        grd.addColorStop(1,"white");
	        ctx.fillStyle=grd;
		ctx.fillRect(0,0,50,50);
	}
</script>
