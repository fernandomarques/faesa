O que é responsividade ?

  • Desenvolvimento responsivo permite criar páginas web que funcionam bem em qualquer dispositivo
  • Não é um programa ou javascript
  • É necessário para acessar as páginas de um dispositivo móvel
  • Exemplo de Design Responsivél com um CSS simples
  • Outra forma é usando um Framework
  • Outra forma é usando um Framework2

Viewport

  • O que é? É a parte visível pelo usuário
  • A viewport varia dependendo do dispositivo e é menor com dispositivos móveis
  • Para consertar alguns navegadores antigos diminuiam a página para caber na tela
  • A tag meta no html5 permite escolhermos o viewport
  • width=device-width: faz com que a largura da página seja a largura do dipositivo
  • Evitem usar elementos com largura grande e fixa
  • Não deixe o conteúdo depender de uma largura específica

Media Query

  • Media query é uma técnica inserida no CSS3
  • Permitindo aplicar uma propriedade apenas se uma condição for verdadeira
  • Exemplo Simples
  • Assim, conseguimos usar um breakpoint no exemplo do slide passado.
  • @media only screen and (orientation: landscape) permite diferenciar entre a orientação da tela
  • Permite saber largura e comprimento do viewport e do dispositivo, a orientação e muito mais

Imagens

  • Se a width da imagem for em porcentagem, ela mudará de tamanho com a tela
  • Ao usarmos max-width a imagem não ficará maior que seu tamanho real
  • Página exemplo com imagem
  • background-size pode ser usado para mudar o tamanho da imagem de fundo. %, contain e cover
  • Com Media também possível mostrar figuras diferentes
  • Também temos o elemento do html5 picture
