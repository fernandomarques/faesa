Tabelas

  • O elemento principal de uma tabela é o table
  • Cada linha da tabela é feita com tr (table row)
  • Cada cabeçalho é feito com th (table header)
  • Por padrão o cabeçalho é negrito e centralizado
  • Um dado da tabela é definido por tr (table data)
  • Assim uma tabela table linhas tr e cada linha é um cabeçalho th ou possui um ou mais dados td

Tabelas - Atributos

  • Align, bgcolor, cellpadding, cellspacing, frame, rules, summary e width não são mais suportados em html5
  • border "0" ou "1" diz se há borda na tabela
  • sortable indica que é possível ordenar a tabela (html 5.1!)
  • A tag caption adiciona uma legenda a tabela

Tabelas - outros atributos

  • Em td ou th podemos usar colspan para dizer a largura da coluna
  • Também podemos fazer isso com rowspan
  • <table>
	<tr>
		<td rowspan="2">Valor</td>
		<td colspan="3">Valor</td>
	</tr>
</table>

Lista

  • Em html podemos ter listas ordenadas ou não ordenadas
  • As lista ordenadas vão aparecer com o número na frente
  • Já as não ordenadas aparecem com uma bolinha
  • As tags são ul e ol
  • Dentro da lista usaremos um item de lista (li)

<ul>
	<li> Elemento 1</li>
	<li> Elemento 2</li>
	<li> Elemento 3</li>
</ul>
<ol>
	<li> Elemento 1</li>
	<li> Elemento 2</li>
	<li> Elemento 3</li>
</ol>
  • Elemento 1
  • Elemento 2
  • Elemento 3
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Listas Ordenadas

  1. O atributo type permite escolher como serão as listas ordenadas
  2. type="1" números
  3. type="I" ou "i" números romanos
  4. type="A" ou "A" letras
  5. O atributo start permite escolher em qual valor inicia
  6. O atributo reversed (html5) faz contagem decrescente

Listas Não Ordenadas

  • É possível alterar o marcador da lista não ordenada utilizando css
  • Para os curiosos: list-style-type:disc|square|circle|none"
