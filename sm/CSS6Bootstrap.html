Grids

  • Sistema de Grid
  • Tamanhos: xs → sm → md → lg
  • Podemos fazer grids aninhados
  • As linhas devem estar dentro de um div "container" ou "container-fluid"
  • Usamos row para criar grupos horizontais de colunas
  • Se o número passar de 12, passa para a linha de baixo
