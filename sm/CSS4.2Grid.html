O que é CSS Grid Layout?

  • É o melhor sistema de layout no CSS até o momento
  • Contrário do flex, é bidimencional
  • Funciona muito bem em conjunto com o flex!
  • A maioria dos navegadores passou a dar suporte em 2017

Elementos

  • Grid Container
  • Grid Item
  • Grid Line
  • Grid Track (espaço entre duas linhas)
  • Grid Cel
  • Grid Area

Container 


.container {
	grid:flex;
	grid-template: 25px 1fr 25px / 1fr repeat(3,25px);
	grid-columns-gap: 5px;
	grid-rows-gap: 10px;
	/* grid-gaps: 10px 5px; */
	justify-items: start | end | center | stretch;
	align-items: start | end | center | stretch;
	/* place-items: start / start */
	justify-content: start | end | center | stretch | space-around | space-between | space-evenly;	
	align-content: start | end | center | stretch | space-around | space-between | space-evenly;
	/* place-content: start / start */
}

Animações

  • gap, suporte nos navegadores menos safari
  • rows e columns, suporte no firefox

Exemplos

  1. Magica em Centralizar (margin auto)
  2. Itens de tamanho fixo, espalhados horizontalmente (row wrap)
  3. Navegacao
    • Desktop - Alinhado a direita
    • Tablet - Centralizado 800px
    • Celular - Em linhas 500px
  4. Site com cabeçalho, main, rodape e laterais
    • Celular - Em linhas
    • Tablet - Laterais dividem espaço 600px
    • Desktop - main e laterais dividem espaco 800px
