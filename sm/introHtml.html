Introdução

  • A linguagem HTML é uma linguagem de marcação
  • Os protocolos para transmitir HTML é o HTTP(1 e 2) e HTTPS
  • Os documentos HTML são utilizados por programas chamados browser ou navegador
  • Navegador interpreta tags e formatação

Conhecendo as tags

  • A linguagem html é baseada em tags aninhadas
  • As tags são comandos escritos entre <nome>
  • Tags devem ser fechadas com seu equivalente </nome>
  • Também é possível fechar a tag direto com <nome/>

Estrutura de página HTML

  • !DOCTYPE html indica a versão
  • html tag inicial de todo documento
  • head
    • Incluir CSS, meta
    • title: titulo do arquivo
  • body corpo do documento

HEAD

  • title: define o título do documento
  • style: pode ser utilizado para colocar código css
  • link: funciona como um 'include'
  • meta: possui informações que, por exemplo, mecanismos de busca podem usar
  • script: códigos em js
  • Pelo html5 não é necessário colocar html,head e body, mas isso pode causar erro em alguns navegadores
