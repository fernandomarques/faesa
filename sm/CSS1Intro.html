CSS

  • Cascading Style Sheet
  • CSS define como os elementos HTML são exibidos
  • Com CSS podemos evitar retrabalho estilizando diversas páginas web ao mesmo tempo
  • Podemos criar arquivos externos no formato .css
  • Exemplo Layout

CSS - sintaxe

  • A sintaxe do CSS normalmente consiste em um seletor e um bloco de declaração
  • 
h1 {
	color: red;
	text-align: center;
}
  • O seletor pode ser uma tag html
  • Reparem que em css usamos : para atribuição e não =
  • Não é necessário colocar o valor entre ""
  • Já que CSS tem C no nome, devemos usar ;

CSS - Como usar

  • Dentro de uma tag html
  • 
<head>
	  <style>
		  h1 {
				color:red;
				font-family:monospace;
			}
		</style>
</head>
<body>
	  <h1>Olá Mundo</h1>
</body>
  • Internamente
  • 

    Olá Mundo

CSS - Como usar

  • Arquivo .css!
  • No arquivo meu.css
  • 
h1 {
	color:red;
	font-family:monospace;
}
  • No arquivo meu.html
  • 
<html>
<head>
	  <link rel="stylesheet" type="text/css" href="meu.css">
</head>
<body>
	  <h1> Olá Mundo!</h1>
</body>
</html>

CSS - Outros tipos de seletores

  • Como vimos, podemos usar as tags HTML como seletor
  • Também podemos agrupar várias tags
  • 
h1, h2, p {
	color: #005500;
}

CSS - Outros tipos de seletores

  • Outros seletores são o de identificador 
  • #para1 {
	text-align: center;
	color: blue;
}

     Texto azul
    

     Texto normal
  • E as classes
  • .center {
	text-align: center;
	color: blue;
}

     Texto azul
    

     Texto normal
  • Um elemento pode ter mais de uma classe!

CSS - Cores

  • Existem várias formas de especificar cores em CSS
  • Nome: red
  • HEX: #ff0000
  • RGB: rgb(255,0,0)
  • RGBA: rgba(255,0,0.3) (css3)
  • HSL: hsl(0,100%,50%) (css3)
  • HSLA: hsla(0,100%,0.3)(css3)
  • Além do color tambem temos o background-color

CSS - Texto

  • text-align: center|left|right|justify
  • text-decoration: none|overline|line-through|underline
  • text-transformation: uppercase|lowercase|capitalize
  • text-indent:50px
  • letter-spacing:3px
  • line-height:0.8;
  • Mais opções para texto
