nth-child()

  • Podemos alterar apenas o n-ésimo filho de um elemento (lista, tabela, paragrafo, etc)
  • li:nth-child(2){
	background:green;
}
  • Também podemos escolher pares ou impares
  • li:nth-child(odd){
	background: blue;
}
li:nth-child(even){
	background: red;
}
li:nth-child(2n+1){
	background: green;
}
