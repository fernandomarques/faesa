Repetição

  • Repetição é coisa para máquina
  • As máquinas são feitas para repetir as mesmas operações
  • E elas são boas nisso!

Utilizando for

  • O comando for possui 3 campos
    1. Inicialização da variável de contagem
    2. Condição de parada
    3. Atualização da variável

Exemplo For


#include <stdio.h>
int main(){
	int i;
	for(i=0; i < 10 ; i ++){
		printf("%d\n",i);
	}
}
  1. Façam um código que conte de 1 até 6 imprimindo na tela
  2. Façam um código que conte de 6 até 1 imprimindo na tela

Utilizando while

  • O loop while (enquanto) vai se repetir enquanto algo for verdade
  • Possui apenas o campo de comparação
  • ATENÇÃO: É mais comum que aconteçam loops infinitos no while

Exemplo While


#include <stdio.h>
int main(){
	int i = 0;
	while(i < 10 ){
		printf("%d\n",i);
		i += 1;
	}
}

Laços for aninhados


#include <stdio.h>
int main(){
	int i,j;
	for(i=0; i < 10 ; i ++){
		for(j = 9; j >= 0 ; j--)
		printf("%d %d\n",i,j);
	}
}
