Repetição

  • Repetição é coisa para máquina
  • As máquinas são feitas para repetir as mesmas operações
  • Logo... elas são boas nisso!

Utilizando for

  • O comando for possui 3 campos
    1. Inicialização da variável de contagem
    2. Condição de parada
    3. Atualização da variável

Exemplo For


#include 
int main(){
	for(int i=0; i < 10 ; i ++){
		printf("%d\n",i);
	}
}

Utilizando while

  • O loop while (enquanto) vai se repetir enquanto algo for verdade
  • Possui apenas o campo de comparação
  • ATENÇÃO: É mais comum que aconteçam loops infinitos no while

Exemplo While


#include 
int main(){
	int i = 0;
	while(i < 10 ){
		printf("%d\n",i);
		i += 1;
	}
}
