Estruturas - O que são?

  • Vimos armazenamentos em variáveis simples (int, char, float)
  • Também vimos Arrays
  • Uma estrutura é um conjunto de uma ou mais variáveis, agrupdas sob um nome para facilitar referência
  • As estruturas podem conter registros de qualquer tipo de dados
  • Exemplo, notas de alunos

Typedef

  • Variáveis estrutura: struct Nome var;
  • Com typedef podemos nos referir a um tipo como se fosse um tipo base
  • Não é criado um novo tipo, apenas facilita a vida
  • Não se limita a struct typedef int inteiro;
  • 
int main(){
	typedef struct Data { int dia, ano, mes;} DATA;
	struct Data dt_test = { 1, 1 , 2001} ;
	// ou
	DATA dt_test = { 1, 1 , 2001} ;
}

Estruturas Passagem de Parametro

  • 
typedef struct data { int dia, mes, ano} Data;
void ImprimeData(Data dt){
	printf("%2i/%2i/%2i\n", dt.dia, dt.mes, dt.ano);
}

int main(){
	Data dt_test = { 1,1,2016};
	ImprimeData(dt_test);
}
  • A passagem de parâmetro é feita por valor

Estruturas - Ponteiros

  • Caso queiramos trabalhar com ponteiro para estrutura
  • Data *ptr_dt;
  • (*ptr_dt).dia OU ptr_dt->dia

Estruturas - Operações

  • Se x é estrutura, x.m acessa seu membro m
  • Se p é ponteiro para estrutura, (*p).m ou p->m
  • Se x e y são variáveis da mesma estrutura, podemos x = y;
  • Não se pode fazer comparação diretas entre as estururas
