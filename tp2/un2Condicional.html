Fluxograma

If-else

  • O condicional permite criar trechos de código que só são executados quando uma condição ocorre.
  • If ... 
  • 
int main(){
	if(10 < 20){
		printf("10 é menor que 20");
	}
}

If-else

  • A condição Else indica o que deve acontecer caso a condição seja falsa
  • Não é obrigatória, mas precisa de um if!
  • 
int main(){
	if (10 < 20){
		printf("10 é menor que 20");
	}else{
		printf("10 não é menor que 20");
	}
}

If-Else If

  • Caso a primeira comparação não seja verdadeira, ainda podemos fazer mais uma pergunta
  • Assim como o else, também não é obrigatório
  • 
int main(){
	if (10 < 20){
		printf("10 é menor que 20");
	}else if (10 > 20){
		printf("10 é maior que 20");
	}
}

If-else Endadeados

  • Não existe limite para a quantidade de if-else
  • Dentro de um bloco {} if ou else, podem existir mais condicionais

Operador Ternário ?

  • Não é aconselhado o uso por deixar a leitura do código mais complexa
  • condição ? expressão1 : expressão2
  • 
int main(){
	float salario;
	scanf("%f",&salario);
	salario = salario > 1000 ? salario *1.05: salario * 1.07;
}
  • Contrário do if-else o ? retorna um valor
