Copiando Strings

  • Vai copiar a string origem para a string destino
  • char *strcpy(char *dest,char *orig)
  • 
char *strcpy(char *dest, char *orig){
	int i;
	int tam = strlen(orig);
	for(i=0; i <= tam; i++){
		dest[i] = orig[i];
	}
	return dest;
}

Concatenando Strings

  • Coloca a string orig imediatamente após dest. Não adiciona espaço nem outro caracter entre as duas.
  • char *strcat(char *dest,char *orig)
  • 
char *strcpy(char *dest, char *orig){
	int i, len;
	for(i=0, len=strlen(orig); orig[i] != '\0'; i++){
		dest[len+i] = orig[i];
	}
	dest[len+i] = '\0';
	return dest;
}

Contando Caracteres

  • Conta quantas vezes o caracter ch aparece na string s
  • int strcountc(char *s, char ch)
  • 
int strcountc(char *s, char ch){
	int i,conta;
	for(i=conta=0; s[i] != '\0'; i++){
		if(s[i] == ch)
			conta++;
	}
	return conta;
}

Outras funções da string.h
