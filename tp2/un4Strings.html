  • "String" 'Caracter'

Strings

  • String não é um tipo básico em C, assim, não conseguimos usar diretamento operadores
  • A declaração de de string é uma simples declaração de vetor
  • 
int main(){
	char nome[30];
	nome[0] = 'Z';
	nome[1] = 'e';
	printf("%s\n",nome);
	nome[2] = '\0';
	printf("%s\n",nome);
}
  • Precisamos usar um delimitador de fim na nossa string

String - Carga Automática

  • 
int main(){
	char nome[20] = "Aula";
	char nome[20] = {'A','u','l','a'};
	char nome[] = "Aula";
	char *nome = "Aula"
}
  • Quando é feita a carga automática, o delimitador é adicionado automáticamente

Leitura e Escrita de Strings

  • O primeiro parâmetro da função printf já é uma string
  • Também podemos passar uma string nos parametros seguintes, basta usar %s
  • A função printf só imprime valores até o \0
  • Também podemos usar a função puts para imprimir uma string
  • Para ler também usamos %s no scanf. Mas não é necessário o &
  • Scanf só lê uma palavra, parando quando encontra espaçõ, tab ou enter
  • Já a função gets pega todos os caracteres

Strlen e passagem de string para função

  • Assim como vimos vetores, a passagem de string também é por referência
  • A função strlen é utilizada para saber o tamanho de uma string
  • 
int strlen(char *s){
	int i = 0;
	while(s[i] != '\0')
		i++;
	return i;
}
