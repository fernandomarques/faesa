Conceitos básicos de função


int Maior(int a, int b){
	return (a > b) ? a : b;
}
int main(){
	printf("O maior entre %d e %d eh %d",10,15,Maior(10,15));
}
Nome da Função Maior
Tipo de Retorno int
Lista de Parâmetros int a, int b
Corpo da função return (a>b) ? a : b;

Tipo de Retorno

  • Como já vimos, uma função possui apenas um tipo de retorno
  • Isso pode ser um problema, caso seja necessário retornar mais de um valor
  • Imaginem uma função que calcula o número de elementos menores que 0 em um vetor de reais e também esses exemplos

Trocando elementos

  • Uma operação que realizamos com frequencia é a de troca, por exemplo para ordenar um vetor
  • Para trocar o conteúdo de dois copos precisamos de um terceiro
  • O mesmo acontece para trocar o conteúdo de 2 variáveis
  • Como podemos fazer essa função?
  • void troca(int a, int b){
 int temp = a;
 a = b;
 b = temp;
}

Passagem de Parâmetros por Valor

  • É enviada uma cópia do valor e não a variável ou expressão
  • Assim, podemos alterar os valores que o valor original não é modificado
  • Quando a função termina, as cópias são apagadas
  • Foi o que fizemos na função que troca valores, por isso ela não funciona!

Passagem de Parâmetros por Referência

  • Na passagem de parâmetros por referência não é enviado o valor mas sim a própria varíavel ou uma referência
  • Assim, qualquer alteração corresponde a uma alteração no valor
  • Para fazer isso em C, precisamos da utilização de ponteiros
  • void troca(int *a, int *b){
 int temp = *a;
 *a = *b;
 *b = temp;
}
  • Sempre que passarmos vetores, estamos, na verdade, passando seu endereço

Quando usar endereços?

  • Se for vetor, nunca
  • Se não for vetor, devemos usar endereço se ela deve ser alterada
  • Implemente a função Calc(v,num, xmin, xmax)
    • v é um vetor de floats
    • num o número de elementos em v
    • xmin e xmax vão armazenar o maior e menor valor no vetor

Passar parametros em main

  • 
main(int argc, char *argv[]){
	int i;
	for(i=0; i < argc; i++)
		printf("%d - Parâmetro = %s\n",i+1;argv[i]);
}
  • argc - um inteiro que indica quantos argumentos foram passados
  • argv - todas strings passadas na linha de comando

Recursividade

  • Recursividade é quando uma função chama a ela mesma, direta ou indiretamente
  • int fat(int n){
if (n == 0)
	return 1;
else
	return n*fat(n-1);
}
  • Sempre ter um critério de parada
  • Depois do critério de parada, é feita a chamada recursiva
  • Implemente a função Down que imprime os N números até 0
