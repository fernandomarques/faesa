Ponteiros e Memórias - Conceito Básico

  • Valor da variável
  • Endereço da varíavel
  • Quase tudo do computador passa pela memória RAM
  • Podemos ver a memória RAM como um vetor de bytes consecutivos
  • É mais fácil referenciar uma variável do que um endereço de memória
  • Ponteiro armazena o endereço de memória de outra variável

Ponteiros - Declaração

  • Para declarar um ponteiro utilizamos int *ptr
  • Para inicializar o ponteiro podemos usar a constante simbólica NULL
  • Para atribuir o endereço de uma variável a um ponteiro, utilizamos &nome_variável
  • int main(){
	int a=5, b=7;
	int *ptr = NULL;
	ptr = &a;
	printf("%d\n",a);
	printf("%d\n",*ptr);
	ptr = &b;
	printf("%d\n",*ptr);
	*ptr = 20;
	printf("%d\n",*ptr);
	printf("%d\n",b);
}

Aritmética de Ponteiros

  • Podemos incrementar um ponteiro, mas ele não será deslocado apenas 01
  • Também podemos decrementar
  • Diferença mostra quantos elementos existem entre dois intervalos
  • Também podemos fazer comparações
  • ptr[2] é igual a *(ptr+2)
  • O nome de um vetor corresponde ao endereço do primeiro elemento

Passagem de Ponteiros em Função

void incrementa(int *a){
		*a +=1;
}
int main(){
	int a = 5;
	incrementa(&a);
	printf("%d\n",a);
}

Ponteiro de Ponteiro

  • Como as outras variáveis, os ponteiros também são armazenados em memória
  • Logo... também possuem endereço!
  • Assim, se armazenarmos o endereço de um ponteiro, temos um ponteiro de ponteiro
  • int main(){
	int x =5;
	int *ptr_x;
	int **ptr_ptr_x;
	ptr_x = &x;
	ptr_ptr_x = &ptr_x;
	printf("x = %d - &x = %ld\n",x,&x);
	printf("x = %d - &x = %ld\n",*ptr_x,ptr_x);
	printf("x = %d - &x = %p\n",**ptr_ptr_x,*ptr_ptr_x);
}

Cuidados

  • Em ponteiros é ainda mais importante iniciar antes de utilizar
  • int *ptr = NULL;
  • O programa pode travar se for atribuído um valor a um ponteiro ainda não inicializado
