Facilitando mudanças no tamanho

  • Como já citado, uma das formas é usando define
  • 
#define tam 10
int main(){
	int vet[tam];
}
  • Também podemos usar const para isso
  • 
const int tam = 10;
int main(){
	int vet[tam];
}
  • Qual a diferença dos dois métodos?

Vetores Multidimensionais

  • Vimos como utilizar vetores com apenas uma dimensão
  • Vetores podem possuir qualquer número de dimenções
  • tipo vetor[dim1] [dim2] ... [dimn]

Declaração de Matriz

  • Matriz nada mais é que um vetor de duas dimenções
  • Essas dimenções podem ser vistas como x,y ou linha e coluna
  • Para declarar um vetor 3x3
  • 
const int dim = 3;
int main(){
	int matriz[dim][dim]; /* temos uma matriz com 3 linhas e 3 colunas */
}

Inicialização de Matriz

  • Como então iremos iniciar uma matriz?
  • 
#define dim 3
int main(){
	int mat[dim][dim] = {}; // Será que funciona???
}

Inicialização de Matriz

  • E nesse caso, o que acontece?
  • 
#define dim 3
int main(){
	int mat[dim][dim] = {1,2,3,4,5,6,7,8,9};
}

Inicialização de Matriz

  • E nesse caso, o que acontece?
  • 
#define dim 3
int main(){
	int mat[dim][dim] = {{1,2,3} , {4,5,6} , {7,8,9}};
}

Indices de uma matriz

mat[0][0] mat[0][1] mat[0][2]
mat[1][0] mat[1][1] mat[1][2]
mat[2][0] mat[2][1] mat[2][2]

Escrevendo em uma matriz

  • Para Escrever valores em uma matriz, iremos usar 2 loop, um para as linhas e outro para as colunas
  • 
int main(){
...
	for(i=0;i < dim; i++){
		for(j=0; j < dim;j++){
			mat[i][j] = i + j;
		}
	}
}
  • Quais os valores dos elementos mat[2][2], mat[1][1] e mat[0,2]

Imprimindo uma matriz

  • Segue um truque para Imprimir a matriz em um formato comum, quebre a linha apenas no loop externo
  • 
int main(){
	// ...
	for(i=0; i < dim; i++){
		for(j=0; j < dim; j++){
			printf("%d\t",mat[i][j]);
		}
		printf("\n");
	}
}
