Características de uma função

  • Cada função tem nome único
  • Uma função pode ser invocada por outras funções
  • Deve realizar uma única tarefa bem definida
  • Deve ser o mais genérico possível para ser reutilizada
  • Recebe parâmetros que alteram seu comportamento
  • Podem retornar valor como resultado dos trabalho

Como funciona a função

  • O código só é executado se a função for chamada
  • Sempre que é executado, o resto do código fica suspenso
  • O programa que invoca usa argumetos que são recebidos pela função
  • Depois de terminar a função pode devolver um valor
  • 
void imprimeX(){
	printf("X\n");
}
int main(){
	imprimeX();
	imprimeX();
	imprimeX();
}

Cabeçalho e Corpo de uma função

  • O corpo de uma função é constituído por códigos em C
  • O corpo da função é seguido por chaves imediatamente após o cabeçalho
  • Não há limite ao número de linhas dentro de uma função

Passagem de parâmetros

  • Os parâmetros são separados por vírgula
  • É necessário os parâmetros tenham seu tipo
  • A passagem de parâmetro deve ser feita entre os parênteses, separado por virgula
  • Também podemos usar uma expressão como argumento de função
  • printX(int quantos){
	int i;
	for(i=0;i < quantos;i++)
		printf("X\n");
}
int main(){
	printX(3);
	printf("\n");
	printX(4);
}

Valor de retorno

  • Instruções que façam algum cálculo retornam um valor
  • Vimos a função sin,cos e pow que retornam valores
  • Quando a instrução return é executada, a função termina e volta ao programa que a invocou
  • O return na main, faz com que ela termine
  • float areaRetangulo(float base,float altura){
	float area;
	area = base * altura;
	return area;
}

O tipo void

  • void em inglês significa vazio
  • Essa palavra reservada serve para dizer que a função não devolve qualquer tipo
  • Uma função que retorna void pode ser chamada de procedimento

Protótipos de funções

  • As funções podem ser colocadas em qualquer parte do código
  • Mas, se for colocada depois da função que a chama, teremos erro
  • Para evitar isso, podemos declarara função colocando apenas o seu cabeçalho seguido de ;
  • Depois, onde for mais conveniente, definimos a função, novamente com o cabeçalho mas agora com as chaves
  • Essa declaração é chamada de protótipo, é uma boa prática declarar todas as funções no início do código

Variáveis locais

  • Também podemos declarar variáveis dentro de funções
  • Depois que uma função termina, todas suas variáveis são destruídas
  • Evitem o uso de variáveis globais, aquelas que não ficam dentro de nenhuma função (nem mesmo a main)

Exemplo Variaveis


void nada(int a){
	int b = 7;
	printf("a: %d b: %d",a,b);
}
int main(){
	int a, b;
	a = 10;
	b = 5;
	nada(a);
	nada(b);
}
